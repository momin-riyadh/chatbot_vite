## Chat System Features

### UI/UX

- Expand the widget to fullscreen mode
- Minimize widget while preserving chat history
- Close widget
- Responsive design (380x640px default)
- Smooth transitions and animations
- Dot pattern background
- Custom branding with logo support
- Online/offline status indicator

### Messaging

- Real-time message input with auto-resize textarea
- Send messages via button or Enter key
- Message bubbles with timestamps
- User and bot message differentiation
- Typing indicators
- Message history preservation
- Scrollable chat area

### Attachments & Media

- Image attachment support
- Multi-file selection
- Attachment preview thumbnails (grid layout)
- Visual preview before sending
- Remove individual attachments
- Image file type validation
- File metadata handling

### User Interaction

- Clipboard support for copy/paste
- Keyboard shortcuts
- Accessible ARIA labels
- Hover states and visual feedback
- Focus management

### Technical

- Local API integration
- Modular screen system
- File handling without raw bytes in JSON
- Data-URL preview generation
- State management
- Event-driven architecture

### Quality Assurance

- 100% test coverage
- Cross-browser compatibility
- Performance optimization
- Error handling
- Accessibility compliance (WCAG)
